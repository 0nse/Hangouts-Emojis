# Fetching Android images

* Download the latest TTF from Android: https://android.googlesource.com/platform/external/noto-fonts/+/lollipop-release/NotoColorEmoji.ttf
* Run `extract.pl` (installing Font::TTF first)


* CBLC table: http://www.microsoft.com/typography/otspec170/cblc.htm
* CBDT table: http://www.microsoft.com/typography/otspec170/cbdt.htm


* The 20 & 28 pixel images come from GMail
* 20/28px images are **missing some icons** - do not use them
* The 128 pixel versions came from an Android package
* 64px versions are built from the 128
